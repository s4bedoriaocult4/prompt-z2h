# PROMPT INTERATIVO ZERO-2-HERO

-- by 0xBB LABS --

Cansado de ficar horas tentando fazer a IA te ensinar algo e alucinar no meio?  
Seus problemas acabaram...

Framework de hardening pedagógico interativo, projetado para criar fluxos de estudo estruturados e controlados.  
Inclui menu, coleta de entrada, controle de tom, modo rígido de progressão e camadas de segurança.

## Funcionalidades Principais

- **Menu Interativo:** Escolha opções como iniciar, definir tom ou saber mais sobre o sistema.  
- **Coleta de Dados do Aluno:** Nome, área de estudo, nível de conhecimento.  
- **Progressão Rígida:** Ensina um conceito por vez, adaptado ao nível do aluno.  
- **Controle de Tom:** Pode ser sério, neutro ou descontraído.  
- **Menu de Feedback:** Permite entender, revisar, ver exemplos, fazer mini-exercícios ou receber críticas construtivas.  
- **Segurança e Hardening:** Camadas de proteção para manter o fluxo de estudo consistente, sem desvios.  
- **Interatividade Total:** Mantém o aluno engajado e garante aprendizado estruturado sem confusão.

## Como Usar


1. Copie o conteúdo do prompt-z2h.txt
2. Execute o prompt no seu ambiente preferido.  
3. Escolha a opção **1 - Iniciar** para fornecer seu nome, área de estudo e nível.  
4. Ajuste o tom com **2 - Escolher Tom**.  
5. Explore **3 - Sobre** para entender melhor a filosofia e os objetivos do framework.  
6. Siga o menu passo a passo para avançar nos conceitos de maneira organizada.  

Este prompt foi pensado para ser **zero-fricção, zero-alucinação e 100% aprendizado focado**, transformando qualquer sessão com IA em uma experiência pedagógica eficiente e divertida.
