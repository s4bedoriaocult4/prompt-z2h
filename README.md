# Interactive Z2H Prompt  
Framework de hardening pedagógico interativo, projetado para criar fluxos de estudo estruturados e controlados.  
Inclui menu, coleta de entrada, controle de tom, modo rígido de progressão e camadas de segurança.

##  Objetivo  
Fornecer um prompt-base que transforme qualquer LLM em um **professor progressivo**, seguindo etapas rígidas, filtragem por nível do aluno e respostas limitadas por regras definidas.

##  Recursos  
- Menu interativo inicial  
- Coleta guiada de: nome, área de estudo, nível e tom da explicação  
- Estilo adaptável conforme o nível do aluno  
- Hard guardrails de saída  
- Restrições explícitas de estrutura  
- Modular e expansível  

## Como Usar  
1. Abra qualquer modelo (GPT, Claude, Gemini, Llama, Mistral).  
2. Copie e cole o conteúdo de `prompt-z2h.txt`.  
3. Siga o menu inicial para configurar seu ambiente interativo.
