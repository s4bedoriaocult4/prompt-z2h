# PROMPT INTERATIVO ZERO 2 HERO  
by 0xBB Labs

Se você já tentou aprender algo com IA e ela começou a sair do assunto, fica tranquilo. Esse prompt foi criado justamente para evitar essa bagunça.

O Zero 2 Hero organiza o estudo, guia cada etapa e impede o modelo de fugir do roteiro. Ele usa menu, coleta informações do aluno, controla o tom da conversa e mantém uma progressão bem clara. É um jeito simples e direto de estudar com IA sem perder tempo.

## Funcionalidades principais
- Menu interativo para iniciar, ajustar tom ou saber mais  
- Coleta de nome, área de estudo e nível  
- Explica um conceito por vez, sempre adaptado ao aluno  
- Controle de tom (sério, neutro, leve, etc.)  
- Menu de feedback ao final de cada explicação  
- Regras de segurança para manter tudo no fluxo certo  
- Experiência interativa, fácil e sem confusão

## Como usar
1. Copie o conteúdo do prompt  
2. Cole no modelo de IA que quiser usar  
3. Escolha a opção **1 - Iniciar** para informar nome, área e nível  
4. Mude o tom com **2 - Escolher tom** se quiser  
5. Veja **3 - Sobre** para entender o propósito do framework  
6. Siga o menu a cada etapa e vá avançando nos estudos

Esse prompt foi feito para evitar alucinações, manter a conversa organizada e deixar o aprendizado muito mais leve e direto.
